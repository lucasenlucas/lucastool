LucasKit (BETA)

De ultieme tool om je vrienden of collega's in de maling te nemen! Kies je besturingssysteem om de installatie-instructies te bekijken.

Installeer LucasKit

Installatie op Linux

1. Open je terminal.

2. Voer het volgende commando uit om het project te clonen:

git clone https://github.com/lucasenlucas/lucaskit.git

3. Ga naar de directory van het project:

cd lucaskit

4. Voer het installatiecommando uit:

sudo ./install.sh

5. Start de tool met:

lucaskit

Installatie op Mac

1. Open je terminal.

2. Voer het volgende commando uit om het project te clonen:

git clone https://github.com/lucasenlucas/lucaskit.git

3. Ga naar de directory van het project:

cd lucaskit

4. Installeer de benodigde dependencies met:

brew install ./install.sh

5. Start de tool met:

lucaskit

Installatie op Windows

1. Open je command prompt (CMD) als administrator.

2. Voer het volgende commando uit om het project te clonen:

git clone https://github.com/lucasenlucas/lucaskit.git

3. Ga naar de directory van het project:

cd lucaskit

4. Voer het installatiecommando uit via PowerShell:

powershell -ExecutionPolicy ByPass -File install.ps1

5. Start de tool met:

lucaskit
